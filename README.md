# Proyecto Métodos Numéricos – Ingeniería de Software

Implementación en Python de cinco métodos numéricos para la resolución de ecuaciones no lineales, con interfaz gráfica y visualización de resultados.

## 📌 Métodos Implementados
- Bisección  
- Falsa Posición (Regula Falsi)  
- Punto Fijo  
- Newton-Raphson  
- Secante  

## 🧩 Estructura del Proyecto
```bash
proyecto_metodos_numericos/
├── metodos/
├── funciones/
├── utils/
├── interfaz/
├── tests/
├── main.py
├── requirements.txt
└── README.md
```
